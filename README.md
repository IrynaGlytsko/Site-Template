# MoGo — Creative & Corporate Portfolio Landing Page

[Посмотреть Live Demo](https://irynaglytsko.github.io/Site-Template/)

Современный, многокомпонентный и адаптивный одностраничный сайт (Landing Page), разработанный на основе популярного креативного шаблона **MoGo**. Проект спроектирован для демонстрации услуг агентства, портфолио команды, отзывов клиентов и последних публикаций блога с акцентом на высокую интерактивность и строгую семантику.

---

## 🚀 Описание проделанной работы

В рамках реализации проекта был выполнен полный цикл фронтенд-разработки по макету с интеграцией интерактивных интерфейсных решений:

### Реализованный функционал и секции:
*   **Главный экран (Intro):** Полноэкранный баннер с приветственным текстом, кнопкой призыва к действию (CTA) и фиксированной навигационной панелью, плавно интегрированной в шапку.
*   **О нас (Story About Us):** Презентационный блок с hover-эффектами для карточек команды и счетчиком ключевых достижений компании в цифрах.
*   **Наши услуги (Services):** Сетка (Grid) с иконками и описанием ключевых направлений деятельности (веб-дизайн, SEO, оптимизация, фотография).
*   **Уникальный дизайн (Unique Design):** Промо-блок с демонстрацией адаптивности под различные типы устройств (мокапы планшета и смартфона).
*   **Интерактивный аккордеон (What We Do):** Реализован раскрывающийся виджет для компактного и структурированного отображения текстовой информации (фотография, креативность, разработка).
*   **Карусель отзывов:** Блок с цитатами клиентов, оформленный в виде интерактивного слайдера с навигационными стрелками.
*   **Портфолио (Some Of Our Work):** Галерея лучших работ агентства с адаптивной плиточной раскладкой и всплывающими плашками при наведении.
*   **Блог (Latest Blog):** Секция с карточками свежих публикаций, включающая графические миниатюры, даты и индикаторы вовлеченности (просмотры, комментарии).
*   **Интеграция карт и подвала:** Секция перехода к карте, развернутый футер с кратким описанием компании, лентой Instagram-публикаций, формой подписки на рассылку и ссылками на социальные сети.

## 🛠 Технологический стек

Проект выполнен с использованием чистого (Vanilla) стека технологий, что гарантирует максимальную скорость загрузки и отсутствие лишних сторонних зависимостей:

*   **HTML5:** Полностью семантическая разметка (использованы теги `<header>`, `<nav>`, `<section>`, `<article>`, `<footer>`). Это обеспечивает высокую SEO-доступность сайта и правильное чтение скринридерами.
*   **CSS3 (SCSS / Flexbox / Grid):**
    *   **Flexbox и Grid Layout:** Использовались для построения сложных сеток портфолио, команды и иконок услуг.
    *   **Препроцессор SCSS:** Логика стилей разбита на модули, используются переменные для цветов/шрифтов и миксины для медиазапросов.
    *   **Плавные анимации:** Переходы (transitions) и эффекты наложения (overlay) при наведении на карточки сотрудников и элементы портфолио реализованы средствами CSS.
*   **JavaScript (ES6+):**
    *   Написание логики работы мобильного меню (бургер-меню).
    *   Управление поведением раскрывающегося аккордеона в секции услуг.
    *   Логика переключения слайдов в блоке отзывов.
*   **GitHub Pages:** Использован в качестве хостинга для публикации веб-ресурса в открытом доступе.

## 📈 Особенности проекта и оптимизация

*   **Адаптивность (Responsive Design):** Интерфейс динамически перестраивается под любые разрешения экранов — от мобильных телефонов до широкоформатных мониторов.
*   **Pixel Perfect & UI/UX:** Точное следование геометрии макета, выверенные отступы, контрастная типографика и отзывчивые интерфейсные элементы для максимального удержания внимания пользователя.

## 💻 Локальное развертывание

Чтобы запустить этот проект локально на своем компьютере:

1. Склонируйте данный репозиторий:
   ```bash
   git clone https://github.com
   ```

2. Перейдите в рабочую директорию проекта:
   ```bash
   cd Site-Template
   ```

3. Откройте файл `index.html` прямо в браузере или запустите локальный сервер (например, расширение *Live Server* в VS Code).
